🚀 КАК ИСПОЛЬЗОВАТЬ:

1️⃣ ПОЛУЧИТЕ ТОКЕН:
   → Перейдите на https://sunoapi.org/ru/billing
   → Выберите тарифный план (от $9.90/месяц)
   → Получите API токен

2️⃣ НАСТРОЙТЕ В ПРОЕКТЕ:
   → Откройте suno-integration.js
   → Найдите секцию sunoapi
   → Замените 'YOUR_SUNOAPI_TOKEN' на ваш токен
   → Измените enabled: false на enabled: true

3️⃣ ТЕСТИРУЙТЕ:
   → Откройте studio.html
   → Создайте трек длительностью до 4 минут
   → Наслаждайтесь профессиональным качеством!
