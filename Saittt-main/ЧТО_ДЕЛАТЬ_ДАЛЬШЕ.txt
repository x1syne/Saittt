🎉 ОТЛИЧНО! Ошибка "уже существует" - это ХОРОШО!

Это значит, что таблицы уже были созданы ранее.
Теперь просто нужно подключить их к вашему сайту.

═══════════════════════════════════════════════════════════════════════

📋 ШАГ 1: ПРОВЕРЬТЕ ТАБЛИЦЫ (30 СЕКУНД)

  1. В Supabase Dashboard откройте "Table Editor" (слева в меню)
  
  2. Вы должны увидеть 3 таблицы:
     ✅ users
     ✅ user_music_data
     ✅ user_stats
  
  3. Если все 3 таблицы есть - отлично! Переходите к Шагу 2.
  
  4. Если какой-то таблицы нет:
     - Откройте "SQL Editor"
     - Скопируйте содержимое файла: SUPABASE_CHECK.sql
     - Запустите первый запрос (проверка таблиц)

═══════════════════════════════════════════════════════════════════════

📋 ШАГ 2: ПОЛУЧИТЕ API КЛЮЧИ (1 МИНУТА)

  1. В Supabase Dashboard откройте "Settings" → "API"
  
  2. Скопируйте 2 значения:
  
     ┌────────────────────────────────────────────────────────────┐
     │ Project URL:                                               │
     │ https://xxxxxxxxxxxxx.supabase.co                          │
     └────────────────────────────────────────────────────────────┘
     
     ┌────────────────────────────────────────────────────────────┐
     │ anon public ключ:                                          │
     │ eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.ey...                 │
     └────────────────────────────────────────────────────────────┘
  
  3. СОХРАНИТЕ ЭТИ ДАННЫЕ в блокнот!

═══════════════════════════════════════════════════════════════════════

📋 ШАГ 3: ОБНОВИТЕ DASHBOARD.HTML (2 МИНУТЫ)

  1. Откройте файл: Saittt-main/dashboard.html
  
  2. Нажмите Ctrl+F (поиск)
  
  3. Найдите: YOUR_SUPABASE_URL
  
  4. Вы увидите эти строки:
  
     const SUPABASE_URL = 'YOUR_SUPABASE_URL';
     const SUPABASE_KEY = 'YOUR_SUPABASE_KEY';
  
  5. Замените на ваши данные:
  
     const SUPABASE_URL = 'https://xxxxx.supabase.co';
     const SUPABASE_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';
  
  6. Сохраните файл (Ctrl+S)

🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

  Проблема: Не вижу таблиц в Table Editor
  
  Решение:
    1. Откройте "SQL Editor"
    2. Вставьте код из файла: SUPABASE_CHECK.sql
    3. Запустите первый запрос
    4. Вы увидите список таблиц

  ───────────────────────────────────────────────────────────────────

  Проблема: Dashboard не показывает данные
  
  Решение:
    1. Откройте консоль браузера (F12)
    2. Проверьте ошибки
    3. Убедитесь, что вы заменили YOUR_SUPABASE_URL и YOUR_SUPABASE_KEY

  ───────────────────────────────────────────────────────────────────

  Проблема: "Supabase is not defined"
  
  Решение:
    Проверьте, что в dashboard.html есть строка:
    <script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>
